# Dyr

Command line to list files on folder.

### How to install

```
deno install --global --allow-net jsr:@rkvcs/dyr
```

### How to use

```
dyr <option>

Options:
  -s, --search      <term>  - Find files or folders by name              
  -d, --only-dirs           - List only directories                      
  -f, --only-files          - List only files 

```
